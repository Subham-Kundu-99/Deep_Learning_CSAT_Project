# Deep_Learning_CSAT_Project
This project focuses on analyzing an e-commerce customer support dataset and building predictive models to understand patterns in customer service performance. The main goal of the project was to explore the dataset, preprocess the data properly, test different hypotheses, and build machine learning models that can make reliable predictions based on the available features.

The project started with data exploration and visualization to understand the structure and characteristics of the dataset. Various exploratory data analysis techniques were applied to identify relationships between variables and detect possible trends in the data. Visualizations such as pair plots and other charts helped in understanding how different features interact with each other. These initial analyses provided useful insights that guided the next steps of the project.

After understanding the dataset, several data preprocessing and feature engineering techniques were applied. Missing values were handled using suitable imputation techniques to ensure that the dataset remained consistent and usable for modeling. Outliers were identified and treated to reduce their impact on the model’s performance. Categorical variables were converted into numerical form through encoding methods so that machine learning algorithms could process them effectively. Additionally, scaling techniques were applied to numerical features to ensure that all variables were on a similar scale, which helps improve the performance and stability of many machine learning algorithms.

The project also involved text preprocessing and normalization steps where necessary. Techniques such as expanding contractions, converting text to lowercase, removing punctuation, eliminating stopwords, tokenization, and lemmatization were used to clean and standardize textual data. These steps help reduce noise in the dataset and make the information more useful for analysis and modeling.

After preprocessing, feature selection and transformation were performed to reduce unnecessary or highly correlated variables. This helps improve model efficiency and reduces the risk of overfitting. The dataset was then divided into training and testing sets using a suitable data splitting ratio so that the models could be trained on one portion of the data and evaluated on unseen data.

To build predictive models, three different algorithms were implemented. The first model was Linear Regression, which served as a baseline model to understand the basic relationship between the features and the target variable. The second model used was Random Forest Regressor, which is an ensemble learning method capable of capturing complex relationships in the data by combining multiple decision trees. The third model implemented was a deep learning approach using an Artificial Neural Network, which can learn complex patterns through multiple layers of neurons.

To further improve model performance, hyperparameter optimization techniques such as GridSearchCV were used. This method tests different combinations of model parameters and selects the ones that produce the best results. The models were evaluated using performance metrics such as R² Score and Mean Squared Error (MSE). These metrics help measure how accurately the model predicts the target variable and how large the prediction errors are.

After comparing the performance of all three models, the Random Forest Regressor showed the best results with higher prediction accuracy and lower error. Because of its strong performance and ability to handle complex relationships in the dataset, it was selected as the final model for prediction.

Overall, this project demonstrates how proper data preprocessing, feature engineering, statistical analysis, and machine learning techniques can be combined to extract meaningful insights and build predictive models. Such models can help organizations make more informed decisions by understanding patterns in their customer support data and improving overall service performance.
